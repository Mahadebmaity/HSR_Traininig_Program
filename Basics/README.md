# HTML

- HTML stands for **Hyper Text Markup Language**
- It is a markup language (It uses markup or tags to define the structure and presentation of text)
- It is used for creating web pages
- It only helps in creating structure of the website
- HTML consists of a series of elements
- HTML elements tells the browser how to display the content

### HTML - Structure 
### CSS - Beautification
### Javascript - Functionality

- Each page on a website is called HTML document

- Each html document starts with <html> and ends with </html>

- All HTML documents start with a document type declaration: <!DOCTYPE html>

- The visible part of the HTML document is kept between <body> and </body>

- **Attributes** - This is being used for providing additional information to the HTML element. It allows us to configure and adjust the behavious to meet specific criteria. It always used in key value pair separated with equal to symbol.

- Meta tags are snippets of HTML code that provide meradata about a web page. They are placed in <head></head> section of a HTML document.
- Meta tags are help in defining the character set, page description, keywords, author, and viewport settings.

## Attributes

- All elements can have attributes
- It provides additional information about elements
- It always specified in the start tag
- It usually comes with key/value pair "name =value"
- Value must be kept in quotation marks(single/double)

### Headings

- In HTML headings are titles or subtitles that we want to display on webpage.
- In HTML 6 levels of headings are present starting from *h1* to *h6*. In which *h1* is the biggest and *h6* is the smallest.

### Paragraphs

- A paragraph always starts on a new line, and is usually a block of text.

- The HTML <pre> element defines preformatted text.

- The text inside a <pre> element is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.

- <hr> tag is being used for horizontal lines.
- <br> tag is being used for line breaks.

- Some tags are empty tags which do not require end tag.

## Text Formatting

- Bold - <b></b>
- Italic - <i></i>
- Underline - <u></u>
- Subscript - <sub></sub>
- Superscript - <sup></sup>
- Marked Text - <mark></mark>
- Strike Text - <s></s>

### Quotation

- Blockquote- It defines a section that is quoted from another source.

- Quoation -  It is used for short quotation

- Abbreviations - It is used for defining abbreviation or an acronym 

- Address - It is used for displaying contact information
- Cite - It is used for defining title of a creative work.
- Bi-Directional Override 

### Comments

- It is used for helping other developer to understand the code, and it is not getting displayed in the browser.
